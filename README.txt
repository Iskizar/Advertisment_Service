Для того чтобы собрать и запустить программу требуется:
1) Скачать Docker Desktop. Скачать можно с официального сайта "https://www.docker.com/"
2) Если были внесены изменения, то требуется пересобрать проект при помощи maven. Сделать это можно при помощи команды mvn clean package или при помощи специальной утилиты maven, находящейся в ide.
3) Далее, при помощи терминала (cmd) нужно перейти в папку FinalProject (Туда, где находятся файлы Dockerfile и docker-compose). Затем нужно написать и запустить команду "docker-compose up". Далее все необходимые ресурсы будут скачаны и программа будет запущена.

Если требуется изменить какие-то параметры программы, то сделать это можно или в Dockerfile и docker-compose или в "FinalProject\src\main\resources\application.properties".
